# ⚔️ W.A.R H.A.M.S — The Battle for Planet X

A sci-fi board game of corporate warfare, territorial control, and tactical betrayal for **2–4 players**.

Each player commands a private military force — squads of heavily armed soldiers known as **H.A.M.S** — deployed by a rival megacorporation to conquer a distant, resource-rich alien world.

## 🎮 Game Overview

- **61-hex planetary board** with randomized tile placement
- **5 resource types:** Oil, Electricity, Intelligence, Industry, Local Favor
- **20 unique BAC (equipment) types** with physical magnetized modules
- **24 Conspire card types** split into Combat and Tactical categories
- **3 victory conditions:** Spaceport Domination, Military Supremacy, Dominance Points
